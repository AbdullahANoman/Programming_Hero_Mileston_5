Question-1 : What is DOM and what is the purpose of DOM?

Answer :

The DOM defines a standard for accessing documents . DOM sytands for document object model It is a programming interface that allows us to create ,change , or remove elements for the document.

Question-3 : What is event bubble?

Answer :

Event bubbling is a type of DOM event propagation.It happens when an element receives an event, and shows lower to upper.

Question-4 : What is a callback function and why will you use it?

Answer :

A callback function is a function passed into another function as an argument This technique allows a function to call another function. A callback function can run after another function has finished.
it executes in the background while the rest of the code runs.
